# AutoCensoR CPU Only

Windows desktop package for machines without NVIDIA CUDA.

Source code: https://github.com/JHKS24/AutoCensoR

## Download

Download `AutoCensoR_CPU_Only.zip` from GitHub Releases, extract it, then run:

```text
AutoCensoR_CPU_Only\AutoCensoR_CPU_Only.exe
```

## Notes

- Model weights are not included.
- Load a compatible `.pt` model from the app's model controls.
- This build favors compatibility over GPU throughput.

## Integrity

Check `SHA256SUMS.txt` after download if you need to verify the package.
